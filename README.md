# وب‌سایت سه‌بعدی مینیمال مامایی

این پروژه یک سایت استاتیک، فارسی، راست‌به‌چپ، واکنش‌گرا و قابل ویرایش برای موضوع **مامایی، وبلاگ سلامت مادر و فروش دوره «درمان مشکل ناباروری با دکتر ارشدی زاده»** است.

## فایل‌های اصلی

```text
index.html                  ساختار صفحه و سکشن‌ها با کامنت فارسی
css/styles.css              طراحی، رنگ‌ها، ریسپانسیو موبایل/دسکتاپ
js/app.js                   تنظیمات قابل ویرایش، محتوا، وبلاگ، Three.js و Babylon.js
assets/images/              تصاویر AI اختصاصی سایت
assets/models/              محل قرار دادن مدل‌های Blender/Unity با فرمت GLB/GLTF
vendor/three.min.js         کتابخانه محلی Three.js
vendor/babylon.js           کتابخانه محلی Babylon.js
```

## اجرا

ساده‌ترین حالت: فایل `index.html` را در مرورگر باز کنید.

برای اجرای بهتر روی لوکال‌هاست:

```bash
python3 -m http.server 8080
```

سپس این آدرس را باز کنید:

```text
http://localhost:8080
```

## تغییر محتوا

بیشتر متن‌های قابل تغییر در ابتدای فایل زیر هستند:

```text
js/app.js
```

در این بخش‌ها تغییر دهید:

- `SITE_CONFIG` برای تلفن، ایمیل، آدرس، قیمت و توضیح دوره
- `CERTIFICATES` برای مدارک و تخصص‌های مامایی خارج از کشور
- `BLOG_POSTS` برای مقاله‌های بیماری‌ها، علائم و راهکارها

## اضافه کردن مقاله وبلاگ

در آرایه `BLOG_POSTS` یک شیء جدید اضافه کنید:

```js
{
  category: "بارداری",
  badge: "موضوع کوتاه",
  title: "عنوان مقاله",
  summary: "خلاصه مقاله",
  tips: ["راهکار اول", "راهکار دوم", "راهکار سوم"]
}
```

## Blender و Unity

در این نسخه، صحنه‌های سه‌بعدی با **Three.js** و **Babylon.js** ساخته شده‌اند. برای استفاده از مدل‌های Blender/Unity:

1. مدل را در Blender یا Unity طراحی کنید.
2. خروجی `GLB` یا `GLTF` بگیرید.
3. فایل را داخل پوشه `assets/models` بگذارید.
4. در `js/app.js` بخش‌های کامنت‌شده مربوط به Three.js یا Babylon.js را توسعه دهید و لودر GLTF را اضافه کنید.

> نکته: چون این پروژه باید بدون نصب ابزار سنگین اجرا شود، فایل Blender/Unity واقعی داخل خروجی قرار نگرفته؛ اما مسیر و ساختار برای افزودن مدل آماده است.

## تصاویر

تصاویر موجود در `assets/images` با هوش مصنوعی برای همین قالب ساخته شده‌اند و به صورت محلی استفاده می‌شوند تا پیش‌نمایش سایت بدون اینترنت هم ظاهر خوبی داشته باشد.

## نکته پزشکی مهم

مطالب پزشکی این سایت آموزشی هستند و جایگزین معاینه، تشخیص یا درمان توسط پزشک/ماما نیستند. برای علائم خطر مثل خونریزی، فشار خون بالا، تب، درد شدید، تنگی نفس یا کاهش حرکت جنین باید فوراً به مرکز درمانی مراجعه شود.
